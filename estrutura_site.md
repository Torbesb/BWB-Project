# Estrutura do Site de Saúde Mental

## Seções Principais

### 1. Página Inicial (Home)
- Banner principal com mensagem acolhedora
- Breve introdução sobre o propósito do site
- Cards de acesso rápido para as principais seções
- Destaques para conteúdos recentes ou importantes

### 2. Transtornos Mentais
- Visão geral sobre transtornos mentais
- Categorias de transtornos (ansiedade, depressão, etc.)
- Sintomas comuns
- Fatores de risco
- Opções de tratamento
- Profissionais a procurar

### 3. Transtornos Alimentares
- Visão geral sobre transtornos alimentares
- Tipos de transtornos alimentares (anorexia, bulimia, compulsão alimentar)
- Sintomas específicos
- Relação com imagem corporal
- Tratamentos especializados
- Equipe multidisciplinar

### 4. Projeto "Explorando os Efeitos das Redes Sociais na Saúde Mental"
- Descrição do projeto realizado em 2024
- Metodologia utilizada
- Resultados e descobertas
- Impacto das redes sociais nos transtornos alimentares
- Depoimentos e casos de estudo
- Recursos e referências

### 5. Chat Interativo
- Interface de chat amigável
- Opções para conectar com outras pessoas
- Regras de comunidade e segurança
- Moderação e suporte
- Salas temáticas (opcional)

### 6. Quiz Interativo
- Quiz sobre conhecimentos em saúde mental
- Quiz sobre transtornos alimentares
- Quiz sobre uso saudável das redes sociais
- Resultados explicativos e educativos
- Compartilhamento de resultados

### 7. Recursos e Ajuda
- Contatos de emergência
- Linhas de apoio
- Indicações de profissionais
- Material para download
- Links úteis

## Navegação

- Menu principal fixo no topo
- Navegação intuitiva com ícones representativos
- Breadcrumbs para facilitar localização
- Barra de busca para encontrar conteúdos específicos
- Footer com links rápidos e informações de contato

## Hierarquia de Informações

1. Informações gerais e introdutórias
2. Conteúdo detalhado sobre transtornos
3. Recursos interativos (chat e quiz)
4. Materiais de apoio e referências

## Arquitetura de Páginas

- Página inicial (index.html)
- Páginas de categorias de transtornos
- Páginas individuais para transtornos específicos
- Página do projeto de 2024
- Página do chat interativo
- Página do quiz
- Página de recursos e ajuda
- Página "Sobre nós"
